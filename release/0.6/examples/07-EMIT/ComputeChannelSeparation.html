EMIT: Compute required channel separation#

This example shows how you can use PyAEDT to open an AEDT project with an EMIT design and analyze the results to determine the required channel separation for overlapping bands. This example requires Ansys AEDT 2023 R2. Uncomment it and run on correct version.
